Listen for Android
==================

**Listen for Android** 는 사운드 이벤트 감지를 위한 안드로이드 라이브러리 입니다. 
Listen을 사용하기 위해서는 `.dpl` 파일이 필요합니다. 
라이센스를 구입하려면 디플리에 문의해주세요.
